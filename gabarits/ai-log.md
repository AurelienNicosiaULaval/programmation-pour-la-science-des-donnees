## Vérification finale

Avant la remise finale, vérifier que:

- le code généré ou modifié avec aide de l’IA est compris par l’équipe;
- les résultats ont été relancés ou vérifiés;
- les sources de données n’ont pas été inventées;
- les limites sont nommées;
- aucune donnée confidentielle ou sensible n’a été soumise à un outil externe non autorisé.
